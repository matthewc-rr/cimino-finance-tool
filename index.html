🚀 Connect Your Google Sheet

One-time setup ‚Äî credentials are saved locally in your browser.

⚠️
Important: This app requires an OAuth 2.0 Client ID (not a simple API key), because it needs write access to your Google Sheet. You authorize access once via your Google account.

Serving the file: OAuth requires the page to be served over HTTP, not opened as a file:// URL. The quickest way: open a terminal in the same folder and run python3 -m http.server 8080, then visit http://localhost:8080/vendor-manager.html.

Setup Instructions

  1. Go to console.cloud.google.com and create or select a project.
  2. Navigate to APIs & Services ‚Üí Library and enable the Google Sheets API.
  3. Go to APIs & Services ‚Üí Credentials ‚Üí Create Credentials ‚Üí OAuth 2.0 Client ID.
  4. Choose application type: Web application.
  5. Under Authorized JavaScript origins, add http://localhost:8080 (plus your production domain if hosting this elsewhere). Click Create.
  6. Copy your Client ID ‚Äî it looks like 123456789-abc.apps.googleusercontent.com.
  7. Create a blank Google Sheet. Copy its ID from the URL:
    https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
  8. Enter both values below and click Save & Connect.
